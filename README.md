#### 说明
本程序使用Java Swing开发，已基本开发完成，具体实现的功能如下：<br/>
1、象棋所有棋子走棋规则检测；<br/>
2、本地UCCI引擎的调用与结果实时读取；<br/>
3、人机对战功能；<br/>
4、棋谱和引擎通信日志记录功能；<br/>
5、最终输赢判断，<br/>
8、棋盘背景、棋子风格和棋盘坐标的自由更换功能，其中棋盘背景共25套，象棋棋子共104套，棋盘坐标风格共三种；<br/>
